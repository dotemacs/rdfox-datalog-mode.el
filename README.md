# `rdfox-datalog-mode`

Emacs major mode for editing RDFox Datalog `.dlog` files.

## Installation

### Use-package

```elisp
(use-package rdfox-datalog-mode
  :vc (:url "https://github.com/dotemacs/rdfox-datalog-mode.el"))
```

### Manual

Add the directory to your `load-path` and require the package:

```elisp
(add-to-list 'load-path "~/dev/emacs/rdfox-datalog-mode.el")
(require 'rdfox-datalog-mode)
```
